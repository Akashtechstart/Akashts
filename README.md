# 🚀 Ultimate Smart Search

**Ultimate Smart Search** एक लाइटवेट और पावरफुल वेब-आधारित सर्च टूल है जो कई प्लेटफॉर्म्स (Wikipedia, DuckDuckGo, YouTube) से डेटा को एक ही जगह पर लाकर दिखाता है। इसमें **Voice Search** की सुविधा भी दी गई है।

## ✨ फीचर्स (Features)

* 🔍 **Multi-Source Search:** एक ही क्लिक में Wikipedia का सारांश, DuckDuckGo के लिंक्स और YouTube वीडियो देखें।
* 🎤 **Voice Search:** हिंदी (hi-IN) में बोलकर सर्च करने की सुविधा।
* 🖼️ **Image Gallery:** सर्च कीवर्ड के आधार पर ताज़ा तस्वीरें।
* 📱 **Responsive Design:** मोबाइल और डेस्कटॉप दोनों पर बेहतरीन काम करता है।
* 🌑 **Dark Mode:** आंखों के लिए आरामदायक डार्क थीम।

## 🛠️ इस्तेमाल की गई तकनीक (Technologies Used)

* **HTML5** - स्ट्रक्चर के लिए।
* **CSS3** - स्टाइलिंग और लेआउट के लिए।
* **JavaScript (ES6+)** - API फेचिंग और वॉइस रिकग्निशन के लिए।
* **APIs:** Wikipedia REST API, DuckDuckGo API.
